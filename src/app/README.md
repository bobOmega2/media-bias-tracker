# App Folder

This folder contains the Next.js App Router pages.

## Subfolders
- `articles/` — Pages for individual articles and article listing
- `collection/` — Pages for user collections
- `dashboard/` — Pages for metrics and analytics

## Notes
- layout.tsx contains the global layout for the app
- page.tsx is the home page
- Dynamic routes are used for individual articles (e.g., articles/[id]/page.tsx)
